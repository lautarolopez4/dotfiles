# Dotfiles - WIP

This is where I keep all of my dotfiles.

Feel free to copy any of the dotfiles or scripts you find here, as much of it is just copied from others' dotfiles. I wouldn't recommend cloning my entire setup, but instead to try copy certain configs and try to build yours, learning is funny.

 ## Software I Use

- **OS**                            : [Arch Linux, btw](https://archlinux.org/)
- **WM**                            : [i3](https://github.com/FrenzyExists/Nord-dot-files-i3/tree/main/config/i3)
- **Shell**                         : [zsh](https://wiki.archlinux.org/index.php/zsh)
- **Terminal**                      : [Kitty](https://github.com/kovidgoyal/kitty). 
- **Bar**                           : [Polybar](https://github.com/polybar/polybar) 
- **Text Editor**                   : [NvChad](https://github.com/NvChad/NvChad) 
- **Wallpaper Manager**             : [feh](https://wiki.archlinux.org/index.php/feh) 
- **Notify Daemon**                 : [dunst](https://wiki.archlinux.org/index.php/Dunst) 
- **Application Launcher**          : [rofi](https://github.com/davatorium/rofi) 
- **Picture**                       : [sxiv](https://wiki.archlinux.org/index.php/sxiv)
- **Cursor**                        : [nord-oreo](https://github.com/0jdxt/oreo-nord-cursors)
- **Screenshot**                    : [maim](https://github.com/morpheusthewhite/spicetify-themes/tree/master/Nord)
- **Browser**                       : [Firefox](https://www.mozilla.org/)
